Welcome

Nagios Manager is an app designed to help folks who are running one or more Nagios XI servers. It’s an interface to the Nagios REST API, and is here to help you lose less of your mind with the day-to-day grind of managing Nagios. Especially if you have multiple Nagios servers. Specifically, Nagios Manager is here to help you manage:

  • Nagios Servers themselves

  • Users on a given Nagios Server

  • Hosts on a given Nagios Server

  • Hostgroups on a given Nagios Server

(Services may come one day, but right now, the API representation of them is so sub-optimal, I’d need to write an app just to manage that. They’re on the list, but rather far down on the list.)

This help is organized in the same way the app is, by tab.
