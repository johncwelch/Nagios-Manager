Basic App Information

While each tab in Nagios Manager has its own specific function, the overall design of each tab is fairly similar:

Each tab has the same basic four sections:

  • Main Tab bar. This is where you select the function you want to manage, which currently consists of:

  • Server Manager , for managing one or more Nagios Servers

  • User Manager , for managing uses on a given Nagios Server

  • Host Manager , for managing hosts on a given Nagios Server

  • Hostgroup Manager , for managing hostgroups on a given Nagios server

  • The main table. This holds lists of servers, users, hosts, or hostgroups

  • Tab Functions. This is where you interact with the items in the main table for the tab, add new items, or delete existing items.

  • Status Display. This shows additional information on a selected item, or the results of commands.

The Application starts up in a size that will work on any display from a 2013 13” MacBook Pro on up. That’s the minimum size for the windows. You can of course, increase the size as you see fit. 

The first time you select a tab when running Nagios Manager, there may be a slight lag as that tab is performing an initial download fo the information it will display. This is done to allow for greater flexibility (you can have different servers selected in each tab), but over a slow connection, the delay might become more noticeable. If you switch out of a tab, then switch back, there should be no lag, at that point, the tab already has data to work with.

When items are added or deleted, that change is saved to the Nagios server (User/Host/Hostgroup Manager) or the application preferences & settings (Server Manager) immediately. There’s no need to worry about saving your changes. Remember that. I try to do as much safety checking as possible, but if you’re using this, I assume you have some experience as a sysadmin. I expect you to be at least somewhat careful. 

Each tab has its own keyboard equivalent, also available in the Window Menu (for those who hate clicking):

  • ⌘ -1 :  Server Manager

  • ⌘ -2 :  User Manager

  • ⌘ -3 :  Host Manager

  • ⌘ -4 :  Hostgroup Manager

I have also tried to provide at least some mouseover and voiceover functionality in terms of what each control does. If I’ve missed something, please let me know via email. (Address is in the footer of this help.)
