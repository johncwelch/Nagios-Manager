Hostgroup Manager

The Hostgroup Manager tab is where you manage hostgroups on the selected Nagios server. You can:

• Add hostgroups

• Get information about an existing hostgroup

Due to limitations in the Nagios REST API, you cannot add hosts to a hostgroup when you create the hostgroup, or to one that already exists. Nor can you delete a hostgroup. In addition, the amount of useful information you can get on a hostgroup beyond what is displayed in the  main table  consists of the members of the hostgroup. Should the API allow for more functionality and information in the future, that will be added to the app.


Host Manager Tab

As with the other tabs in Nagios Manager, the Hostgroup Manager contains the three basic  functional areas . The  main table  shows the Hostgroup name and active status for every hostgroup on a given Nagios server. The members of the selected hostgroup are shown in the  status display .


Selecting a Server

The “Select Server” popup is where you select the specific server whose hostgroups you are managing. If there are multiple servers in  Nagios Manager’s preferences file , then the first server is decided by its order in the preferences file. The first hostgroup returned for a given server is selected by default in the  main table , and the  status display  will show the members of the selected hostgroup. If you change servers, the  main table  will be reloaded with the hostgroups for the server you select.


Selecting and Getting member information for a Hostgroup

To select a hostgroup, click on that hostgroup’s row in the  main table . The  status display  will automatically display the member hosts, if any exist, for that hostgroup.


Adding a Hostgroup

When adding a hostgroup, the Hostgroup Name and Hostgroup Alias are required. Adding a hostgroup via Nagios Manager does not populate the hostgroup with any hosts, nor can you do this from Nagios Manager due to limitations in the Nagios REST API. There is six-second delay in refreshing the hostgroup list after adding a hostgroup. To manually refresh, re-select the current Nagios Server in the “Select Server” popup.
